# TypeScript — Mid-Level Guide

> Full interactive version available on the [Tech Hub Learning Platform](/language/typescript/mid)

## Topics Covered

- Advanced Generics and Constraints (defaults, inference patterns, Zod integration)
- Utility Types (Partial, Required, Pick, Omit, Record, Exclude, Extract, ReturnType, Parameters, Awaited, NoInfer)
- The `satisfies` Operator (TS 5.0 — type-safe defaults without widening)
- Mapped Types and Key Remapping
- Conditional Types and `infer` Keyword
- Strict Mode and Compiler Flags (`verbatimModuleSyntax`, `isolatedModules`, `noUncheckedIndexedAccess`)
- Discriminated Unions and Exhaustive Checking
- Declaration Files and Module Augmentation
- Testing Typed Code (type-level tests, typed mocks)

## Prerequisites

- Completion of the TypeScript Beginner guide or equivalent knowledge
- Comfortable with interfaces, generics, and union types
- Experience building TypeScript projects with tsconfig

## Estimated Time

45 hours
