# Language Learning Paths

Standalone deep-dive learning paths for core programming languages used across all tech roles.

## Languages

| Language | Levels | Related Roles |
|----------|--------|---------------|
| [Python](Python/) | Beginner, Mid, Senior | AI Engineer, Data Scientist, Data Engineer, ML Engineer, Backend Developer, DevOps |
| [JavaScript](JavaScript/) | Beginner, Mid, Senior | Frontend Developer, Backend Developer, Marketing Tech, QA Engineer |
| [HTML & CSS](HTML-CSS/) | Beginner, Mid, Senior | Frontend Developer, Marketing Tech |
| [SQL](SQL/) | Beginner, Mid, Senior | Data Engineer, Data Scientist, Backend Developer, ML Engineer |
| [TypeScript](TypeScript/) | Beginner, Mid, Senior | Frontend Developer, Backend Developer, Tech Lead/Architect |

## Structure

Each language path contains three levels:
- **Beginner.md** — Foundations and core concepts
- **Mid.md** — Intermediate patterns and techniques
- **Senior.md** — Advanced topics and expert knowledge

## How to Use

1. Start with the language most relevant to your target role
2. Work through Beginner → Mid → Senior progressively
3. Complete the interactive exercises at the end of each section
4. Take the quiz to test your understanding
5. Use the code sandbox examples for hands-on practice

## Relationship to Role Paths

These language paths complement the role-based roadmaps. While role paths teach you *what* to build and *why*, language paths teach you *how* to write the code. Each role overview links to the relevant language deep dives.
